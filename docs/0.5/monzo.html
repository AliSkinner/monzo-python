monzo.monzo module¶

A wrapper around the official Monzo API endpoints.

This module contains the class Monzo which represents a wrapper around HTTP calls to Monzo’s API endpoints.

class monzo.monzo.Monzo(access_token)¶

Bases: object

The class representation of Monzo’s API endpoints.

Please note that functions without a reference to the official Monzo API docs page are convinence functions which are created - based on the official API functions - to make life easier for developers.

e.g. get_first_account calls get_account and returns the first account object, if it exists.

Parameters:access_token – The access token to authorise API calls.
